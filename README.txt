Hola pibes!

1. Necesario tener instalado Node.js.
2. Ejecuté 'npm init -y' para crear el package.json.
3. Instalé Express.js como dependencia con el comando 'npm install express'
4. Cree el 'server.js' para configurar el servidor Express.
5. Instalé la biblioteca sqlite3 con el comando 'npm install sqlite3'.
6. Instalé DB Browser for SQLite para ver la bdd (https://github.com/sqlitebrowser/sqlitebrowser/releases/download/v3.12.2/DB.Browser.for.SQLite-3.12.2-win64.zip)
7. Por ultimo instalé cors con 'npm install cors' para resolver problemas de restricciones entre puertos diferentes.

Modifiqué el script.js para que funcione con esto nuevo y algunas cosas en el form que tiene los inputs en register.html.

¡¡IMPORTANTE!!
Para que funcione la base de datos hace falta poner el comando node server.js en la terminal.
